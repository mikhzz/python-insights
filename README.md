# Análise de Dados em Python - Exemplo Prático

Este repositório contém um exemplo de como abordar a análise de dados em Python, simulando uma situação real em uma empresa fictícia. A análise é realizada utilizando tecnologias como Python, Pandas, Jupyter Notebook e Plotly.

## Conteúdo do Projeto

- **inicial.ipynb**: Jupyter Notebook contendo o código-fonte para a análise de dados.
- **dataset.csv**: Base de dados fictícia utilizada na análise.

## Tecnologias Utilizadas

- **Python**: Linguagem de programação principal.
- **Pandas**: Biblioteca para manipulação e análise de dados.
- **Jupyter Notebook**: Ambiente interativo para desenvolvimento e visualização de código.
- **Plotly**: Biblioteca para criação de gráficos interativos.

## Como Executar o Código

Para visualizar e executar o código deste projeto, siga os passos abaixo:

1. **Navegue até o diretório do projeto:**
    ```bash
    cd exemplo-analise-dados
    ```

2. **Abra o arquivo Jupyter Notebook `inicial.ipynb` para visualizar e executar o código.**
